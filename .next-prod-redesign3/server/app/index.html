AgeGate Proxy
Privacy-preserving age verification
Identity-separated age verification

Verify age eligibility without exposing full identity to the client platform.

AgeGate Proxy sits between the relying platform and the identity provider. The user chooses `.gov` or Gmail, and the client receives only a minimal proof result.

How the proof contract works
01
Client platform requests an age check.
02
User selects a trusted provider.
03
AgeGate validates the provider response.
04
Only the age proof result is returned.
Example response: { verified: true, age_band: "18+" }
Minimal disclosure
Only eligibility, age band, token, and expiry are shared.
Editable demo data
Platforms, providers, users, and logs live in JSON-backed seeds.
Cloudflare-ready MVP
Next.js Route Handlers keep the demo full-stack and ready for Workers deployment.
