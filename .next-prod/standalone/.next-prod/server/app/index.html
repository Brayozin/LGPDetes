AgeGate Proxy
Verificação de idade com privacidade

Comprove maioridade sem expor a identidade para a plataforma cliente.

O AgeGate Proxy fica entre a plataforma e o provedor de identidade. O usuário escolhe `.gov` ou Gmail, e o cliente recebe apenas o resultado mínimo da prova.

Fluxo
01
A plataforma solicita a checagem de idade.
02
O usuário escolhe um provedor confiável.
03
O AgeGate valida a resposta do provedor.
04
Só o resultado mínimo volta para o cliente.
Exemplo de retorno: { verified: true, age_band: "18+" }
Divulgação mínima
Só elegibilidade, faixa etária, token e expiração são compartilhados.
Dados editáveis
Plataformas, provedores, usuários e logs ficam em seeds JSON.
MVP pronto para deploy
As rotas do Next.js mantêm a demo full stack e pronta para Workers.
